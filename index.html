Introduction

Java is a high-level, class-based, object-oriented programming language.

It is designed to have as few implementation dependencies as possible, making it one of the most popular programming languages in the world.

What You Should Already Know

This guide assumes the following:

  • General understanding of programming languages
  • Familiarity with object-oriented programming (OOP)
  • Prior experience with basic programming languages (e.g., Python, JavaScript)
  • Understanding of loops, functions, and conditionals
Java and JavaScript

While Java and JavaScript share similar names, they are fundamentally different programming languages designed for different purposes.

Java is a general-purpose, compiled language, while JavaScript is used mainly for web development.

Hello World

The simplest Java program:


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

This program prints "Hello, World!" to the console.

Variables

Variables are containers for storing data values in Java.

In Java, you declare variables by specifying the data type followed by the variable name.

Declaring Variables

To declare a variable, you do so by the following syntax:

int myNumber = 5;

Here, int is the data type, and myNumber is the variable name.

Global Variables

Global variables are accessible from all parts of the program.

In Java, global variables are declared within a class but outside of any methods.

Constants

In Java, constants are declared using the final keyword.

final int MY_CONSTANT = 10;

This makes sure the value of the constant cannot be modified.

Data Types

Java supports various data types:

  • int
  • double
  • char
  • boolean
If...Else Statement

Conditional logic in Java uses the if and else statements.


if (condition) {
    // block of code
} else {
    // block of code
}
While Loop

The while loop runs a block of code as long as the condition is true.


while (condition) {
    // code to be executed
}
Functions

Functions in Java are declared within classes using methods. For example:


public void myFunction() {
    // code to execute
}
